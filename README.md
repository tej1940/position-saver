This Assetto Corsa mod lets you save your position manually or automatically if enabled on any map of your choice.


	Requirements:
	-Content Manager must be installed.
	-Custom Shaders Patch (CSP) is required.

This project is still in development and is not fully optimized yet.

i also made a car swapper mod here is the link : https://github.com/tej1940/car-swap-assetto-corsa

I'm not completely sure how the default settings will behave on different systems, so I'd really appreciate any feedback. If you run into any issues or have suggestions for improvements, please let me know!
